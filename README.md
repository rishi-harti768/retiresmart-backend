# backend for RetirSmart

## setup guide

1.  install postgreSQL
2.  add environment variables in `.env` file at root directory for project

```
PORT=***
JWT_ACCESS_SECRET=***
JWT_REFRESH_SECRET=***

DB_HOST=***
DB_PORT=***
DB_USER=***
DB_PASSWORD=***
DB_DATABASE_NAME=***
DB_SCHEMA=***

HOST_EMAIL_ADDRESS=***
HOST_EMAIL_PASSWORD=***

```
